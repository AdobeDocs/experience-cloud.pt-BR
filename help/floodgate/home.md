# Descubra o Adobe Target Floodgate {#get-started}


## O que é Floodgate?

O Floodgate é um sistema de gerenciamento de recursos e experimentação que ajuda os aplicativos a fornecer novas experiências de forma segura, gradual e com menos riscos.

Em vez de ativar um novo recurso para todos de uma só vez, o Floodgate permite que as equipes de produtos:

* Liberar recursos gradualmente
Comece com uma pequena porcentagem de usuários e aumente com o tempo. Se algo não se comportar conforme o esperado, o recurso pode ser desativado rapidamente, sem tempo de inatividade ou reinstalação do software.

* Direcionar recursos para públicos-alvo específicos
Habilite novos recursos para grupos selecionados (por exemplo, programas beta, determinadas regiões ou planos específicos) para que as alterações possam ser validadas e refinadas antes de serem amplamente disponibilizadas.

* Executar experimentos e testes A/B
Experimente versões diferentes de um recurso (como layouts alternativos, fluxos ou recomendações) e avalie qual deles funciona melhor, usando dados para orientar as decisões do produto.

* Reduzir o risco de versões críticas
Alterações grandes podem ser divididas em implantações menores e controladas, de modo que os problemas são mais fáceis de detectar, entender e corrigir antes que afetem todos.

* Responda rapidamente aos problemas
Se ocorrer um problema de bug ou desempenho, as equipes poderão desativar somente o recurso afetado, mantendo o restante do aplicativo estável.

Na prática, o Floodgate ajuda os usuários finais a experimentar melhorias mais frequentes com menos interrupções: novos recursos podem chegar antes, ser testados com mais cuidado e ajustados rapidamente com base no feedback do mundo real.
