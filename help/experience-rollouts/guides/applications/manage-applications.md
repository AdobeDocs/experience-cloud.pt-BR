# Gerenciar aplicativos {#manage-applications}

Um **aplicativo** em Implantações de Experiência representa o serviço ou produto que você deseja controlar com sinalizadores de recursos. Antes de criar sinalizadores de recursos, você deve integrar pelo menos um aplicativo ao console.

## Quem pode gerenciar aplicativos {#permissions}

Somente membros com a função **Administrador** podem adicionar ou modificar aplicativos. Se você não tiver essa função, entre em contato com seu administrador.

## Adicionando um aplicativo {#add-application}

Para adicionar um aplicativo, consulte [Integrar o aplicativo](onboard-your-application.md) para obter instruções passo a passo.
