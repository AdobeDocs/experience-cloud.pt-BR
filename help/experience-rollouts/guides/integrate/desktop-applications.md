# Aplicativos de desktop {#desktop-applications}

Os aplicativos de desktop se integram às Implantações de experiência por meio da SDK. Entre em contato com seu representante da Adobe para obter orientação de integração específica para sua pilha de desktops.

>[!NOTE]
>
>A documentação do Desktop SDK está sendo preparada e estará disponível em breve.

## ID do aplicativo {#app-id}

Os clientes da área de trabalho podem usar um **código de produto e uma versão de produto** como o identificador de aplicativo.
